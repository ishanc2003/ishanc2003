## Featured Projects

### 1. Hallucination-Resistant Multi-Hop Q&A (March 2026)
Evidence-based RAG pipeline for multi-hop question answering on HotpotQA. Combines BM25 + FAISS hybrid retrieval, BAAI/bge-reranker-v2-m3 cross-encoder reranking, and citation selection to prevent hallucinated references. Local inference using Qwen 2.5:7B via Ollama. Achieved 62% exact match, 79% answer F1, 83% supporting fact F1.

### 2. Wine Explorer (April 2026) 
A voice-enabled wine discovery web app using FastAPI and vanilla JS, with Whisper speech-to-text, streaming LLM responses via Ollama (Qwen 2.5:7B), and browser-native. Added structured intent extraction (Gemma 2:9B) with regex fallback, embedding caching with MD5 validation, and sentence-buffered voice output for natural-sounding responses, all running locally with zero cloud APIs 

### 3. N.O.V.A - Narrative and Observational Virtual Assistant (Jan 2024)
Local-first AI writing assistant built with Python 3.11, Flask, Vue.js, and Electron. Features offline inference, resource-aware context management, response streaming for sub-5-second latency, and SHA-256 encryption for secure local storage. Patent application filed with Indian Patent Office (No. 202641027824).

### 4. OmniWatch (March 2023)
WearOS smartwatch app using Jetpack Compose and Kotlin with Android Studio. Features multi-face dynamic watch designs, 1,000+ unique visual and interaction configurations, dark-mode rendering for battery optimization, and SoundPool-based audio handling. Deployed via Google Play Closed Testing with 140+ beta testers.

### 5. Go Player (October 2025)
Minimax Go agent for 5x5 board with alpha-beta pruning, fast legal-move generation, and heuristic evaluation (liberties, eyes/connectivity, captures). Achieved ~80% win rate against random, greedy, and Q-learning players.

### 6. Handwritten Text Generator (July 2025)
DDPM-based diffusion model trained on MNIST dataset (85,000 examples) for generating handwritten text from random noise. Pre-trained model with Python menu-driven inference interface, 15 color options, and optimized generation under 7 seconds.
